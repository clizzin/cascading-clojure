= cascading-clojure

A Clojure library for [Cascading](http://cascading.org).

== Hacking

Get [Leiningen](http://github.com/technomancy/leiningen) 1.1.0.

    $ lein deps
    $ lein compile-java
    $ lein compile
    $ lein test

See `src/clj/cascading/clojure/api_example.clj` for an example of how to run a Cascading flow.
